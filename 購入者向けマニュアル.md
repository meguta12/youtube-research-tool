# YouTubeリサーチツール ご利用マニュアル

このたびはご購入ありがとうございます。
このツールはお手元のブラウザで動きます。インストール不要です。

## 1. 起動方法

1. 購入時に届いたメールにある **ツールのURL** をブラウザで開きます（Chrome 推奨）。
2. 購入時のメールに記載の **ライセンスキー** を入力して「認証する」を押します。
3. 初回のみ、3ステップのオンボーディングが表示されます。

ライセンスキーはブラウザに保存されるため、次回以降は入力不要です。

## 2. YouTube APIキーの取得（最初に1回だけ）

このツールは Google が公式に提供している「YouTube Data API v3」を使います。
**無料**で利用でき、毎日10,000ユニットの枠があります（このツールでは1回のリサーチで100〜200ユニット程度）。

### 取得手順

1. [Google Cloud Console](https://console.cloud.google.com/) に Google アカウントでログイン
2. 上部の「プロジェクトを選択」→「新しいプロジェクト」→ 任意の名前で作成
3. 左メニュー「APIとサービス」→「ライブラリ」→ **「YouTube Data API v3」**を検索→「有効にする」
4. 左メニュー「APIとサービス」→「認証情報」→「+認証情報を作成」→ **「APIキー」**
5. 表示された `AIza...` で始まる文字列をコピー
6. ツールのオンボーディング画面 or「設定」画面の「APIキー」欄に貼り付け、
   「✓ 接続テストして保存」を押します。

> APIキーは **あなたのブラウザだけに保存** され、外部に送信されることはありません。

### ⚠️ 重要：APIキーには必ず「制限」をかけてください

このツールはお客様のブラウザだけで動く構造のため、**技術的にはAPIキーを内部に隠しきることができません**
（ブラウザの開発者ツールから確認できる構造です）。万一の漏洩に備え、Googleが推奨する以下の制限を**必ず**設定してください。

これを設定すれば、**万一キーが第三者に渡っても他人には使えません**。

#### 設定手順（5分で完了）

1. [Google Cloud Console の「認証情報」画面](https://console.cloud.google.com/apis/credentials) を開く
2. 作成したAPIキーをクリック
3. **「アプリケーションの制限」** で **「ウェブサイト」** を選択
   - 「ウェブサイトの制限事項」に **このツールのURL** を `/*` 付きで追加
   - 例：`https://your-tool.vercel.app/*`
4. **「APIの制限」** で **「キーを制限」** を選択
   - **「YouTube Data API v3」** だけにチェックを入れる
5. **「保存」** を押す（反映まで数分かかる場合があります）

これで、このキーは「このツール」かつ「YouTube検索」以外には使えなくなります。

### もしAPIキーが漏れてしまったら

1. [Google Cloud Console の「認証情報」画面](https://console.cloud.google.com/apis/credentials) を開く
2. 該当キーの「︙」メニューから **「APIキーを削除」** を選択 → 即時無効化
3. 同じ画面で新しいキーを発行し、ツールの「設定」画面に登録し直してください。

## 3. 基本の使い方

ホーム画面で以下を入力／選択して「▶ リサーチ実行」を押します。

| 項目 | 説明 |
|---|---|
| キーワード | 必須。例：「副業」「在宅ワーク」 |
| 期間 | 今日 / 今週 / 今月 / 3ヶ月 / 全期間 |
| 動画尺 | 全て / 4分未満（ショート寄り）/ 4-20分 / 20分以上 |
| 並び替え | 視聴回数 / 関連度 / 新着 / 評価 |
| 取得件数 | 10 / 25 / 50 / 100 |

実行すると上部のタブで結果が確認できます：

- **動画リスト** … 1動画1行で詳細を確認
- **チャンネル分析** … 同じテーマで複数本ヒットしたチャンネル順
- **競合分析** … タイトルの頻出ワード、投稿曜日・時間帯の傾向
- **サムネ一覧** … 再生数順にサムネがグリッド表示

右上の「⬇ Excelダウンロード」でいつでも保存できます。

## 4. 指標の意味

- **1日平均再生数**：再生数 ÷ 経過日数。新しい動画と古い動画を平等に比較できます。
- **登録者比**：再生数 ÷ チャンネル登録者数。**1.0を超えると登録者数より多く再生された＝バズ寄り**。
- **再現性スコア**：このキーワードでヒットした本数 ÷ チャンネル総動画数。**高いほど安定して再生数を取れている**可能性。
- **🔥 急上昇フラグ**：登録者比が1.0を超えた動画。

## 5. うまく検索できないとき

- **「該当動画が見つかりませんでした」が出る**
  - キーワードを一般的な語に変える（例：「副業 月10万」→「副業」）
  - 期間を「全期間」にする
- **大手公式チャンネルばかり並ぶ**
  - 「設定」→「除外チャンネルID」に該当チャンネルのIDを追加
- **「quotaExceeded」エラー**
  - その日の枠を使い切りました。翌日（太平洋時間0時 ≒ 日本時間17時）にリセットされます

## 6. データの安全性について

- 検索条件・APIキー・検索結果は、すべて **あなたのブラウザの中だけ** に保存されます
- 販売者を含む第三者がデータを見ることはありません
- ブラウザの履歴・Cookieを削除すると保存内容も消えます。Excelダウンロードでバックアップを推奨します
